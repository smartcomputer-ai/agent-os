# Session Core Plan Pack

Defs-only AIR export for reusable session/workspace orchestration core plans.

- Layout follows `air/exports/plan-packs/<pack>/defs.air.json`.
- Exports `core_*` plans for composable-core consumption.
- Includes required supporting `defschema` contracts.
- No manifest node is included.
- Capability slots in exported plans are stable API (`cap_workspace`).
